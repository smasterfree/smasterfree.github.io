##过滤监听端口

1

lsof -i -sTCP:LISTEN

当然也可以使用 grep LISTEN的方法

1

lsof -i | grep -i LISTEN

找到已经建立的连接

关键词 ESTABLISHED

1

lsof -i -sTCP:ESTABLISHED

或者

1

lsof -i | grep -i ESTABLISHED

用户信息

获得某一个用户的信息，包括他们的网络活动，打开文件信息 例如：

1

lsof -u root

排除例外规则

1

lsof -u ^root

结束一个用户的所有活动

1

kill -9 `lsof -t -u daniel`

命令与进程

查看某一命令在使用哪些文件，连接

1

lsof -c python

##查看某一进程ID

1

 lsof -p 10075

某一个命令的pid是多少

1
2

lsof -t -c python
lsof -t -c apache2

文件和目录

1

 lsof /home/daniel/firewall_whitelist.txt

高级用法

和tcpdump相似，也可以加入过滤器

过滤用户与ip

1

lsof -u daniel -i @1.1.1.1

-t 和 -c 来HUP进程

1

kill -HUP `lsof -t -c sshd`

就是把lsof列出的全杀掉

列出 port range里面，打开的连接

1

lsof -i @fw.google.com:2150=2180

lsof大法好，退 netstat 和 ps 保平安！
